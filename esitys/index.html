Tehtäiskö tästä vähän komeempi esitys samontien
Tää olis tulevaisuudessa kannsa aika paljon "ammattimaisempi" työkalu kuin power-vitun-point

Pittää olla node (v10 tai uuempi) ja npm

Asennus:


						$ git clone https://github.com/hakimel/reveal.js.git esitys
						$ cd esitys && npm install
						$ npm start
